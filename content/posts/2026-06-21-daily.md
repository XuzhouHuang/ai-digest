🔥 **今日高光**

1. **「Skills for Real Engineers」冲上 Trending——agent 技能体系开始平民化**  
   今天最值得 builder 记一笔的，是 Matt Pocock 开源的 **skills** 仓库（13.7 万 ⭐，今日 +1,520）。标语很直白：*Skills for Real Engineers. Straight from my .claude directory.* 它把一个真实工程师 `.claude` 目录里沉淀下来的可复用技能直接开源出来。结合昨天 *superpowers*、*mattpocock/skills* 这类项目一起看，趋势已经很明显：**agent 的核心资产正在从「prompt」转向「skills」**——一套可版本化、可共享、可被 agent 直接调用的工程能力包，正在成为 builder 之间新的「公共基础设施」。  
   🔗 <https://github.com/mattpocock/skills>

2. **Kilo：把「agentic engineering」做成一体化平台**  
   **Kilo-Org/kilocode**（2.3 万 ⭐）今日继续冲榜，定位是 *all-in-one agentic engineering platform*——围绕「最流行的开源 coding agent」做 build / ship / iterate 的全链路。这正好接上了昨天 GLM-5「From Vibe Coding to Agentic Engineering」的叙事：**agentic engineering 不再只是一句口号，而是开始有人把它产品化、平台化。** 对中文 builder 来说，这是一个可以认真纳入工具链评估的开源候选。  
   🔗 <https://github.com/Kilo-Org/kilocode>

3. **jcode：用 Rust 写的「Coding Agent Harness」**  
   **1jehuang/jcode**（7.3k ⭐）是今天另一个值得关注的底层项目——一个用 **Rust** 实现的 coding agent harness，topics 直接点名 `claude / cli / coding-agent / llm / mcp`。它和 Kilo 形成有意思的对照：一个往「平台 / 全家桶」走，一个往「轻量、可嵌入的 agent 运行骨架」走。**harness 这一层正在变成 coding agent 生态的兵家必争之地**——谁掌握了 agent 与工具、文件、终端交互的「壳」，谁就掌握了上层体验。  
   🔗 <https://github.com/1jehuang/jcode>

4. **Voicebox：开源 AI 语音工作室，把 TTS/克隆/听写打包**  
   **jamiepine/voicebox**（3.0 万 ⭐）是今天 AI-native 创作工具里最亮的一个：一个开源的 **AI voice studio**，主打「Clone, dictate, create」，技术栈点名 `qwen3-tts / mlx / cuda`——既能跑 Apple Silicon（MLX）也能上 CUDA。对做语音类产品或想在本地集成高质量 TTS 的 builder 来说，这是一个可以直接拿来当底座的开源方案。  
   🔗 <https://github.com/jamiepine/voicebox>

5. **Palmier Pro：为 AI 而生的 macOS 视频编辑器**  
   **palmier-io/palmier-pro**（2.7k ⭐，今日 +904）是一个 *macOS video editor built for AI*，topics 里 `claude / mcp / seedance2` 同时出现——也就是说它把 LLM 编排（claude + MCP）和视频生成模型（seedance2）直接缝进了剪辑流程。配合昨天的 OpenMontage（agentic video production），可以看出 **「AI 原生视频生产」正在从 pipeline 走向真正的桌面级编辑器**，agent 不只是生成素材，而是参与到时间线编辑本身。  
   🔗 <https://github.com/palmier-io/palmier-pro>

💡 **今日观察**

周末没有新论文，但 GitHub Trending 反而把一条主线照得更清楚：**coding agent 的竞争，正在从「模型够不够强」下沉到「壳 / 平台 / 技能」这三层。**

- **壳（harness）**：*jcode* 这类用 Rust 写的 agent 运行骨架，在抢「agent 与工具、终端、文件系统交互」的入口；  
- **平台（platform）**：*Kilo* 把 build/ship/iterate 全链路一体化，试图成为团队级的 agentic engineering 工作台；  
- **技能（skills）**：*mattpocock/skills* 把真实工程师 `.claude` 目录里的能力包开源，预示 **skills 正在取代 prompt 成为 agent 的核心可复用资产**；  
- **应用层**：*Voicebox*、*Palmier Pro* 则说明 AI-native 创作工具（语音、视频）已经走到「可直接当底座」的成熟度。

一个共同的底层事实是：**claude + MCP 几乎成了今天所有上榜 agent 项目的默认装备**。对 builder 来说，今天最值得带走的判断是：**护城河越来越不在「接哪个模型」，而在你有没有一套好用的 harness、可复用的 skills，以及能把它们端到端跑起来的工程平台。**
