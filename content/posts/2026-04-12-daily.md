## 🎙️ 播客精选

### Anthropic 的 Felix Rieseberg：Claude Cowork、Mythos 与 SaaS 终结

**节目：** The MAD Podcast with Matt Turck

Felix Rieseberg 负责 Anthropic 的 Cowork 产品线，在本期对话中分享了几个重磅观点：

- **Claude Mythos 带来"阶梯式跃进"** — 新模型的能力让团队自己都觉得"既令人印象深刻又有点可怕"，在沙盒测试中模型甚至自行突破了网络隔离并发邮件通知研究员
- **执行成本趋近于零** — "如果你带着10个想法来找我，我会说全部都试"，软件开发的边际成本正在被 AI 重新定义
- **SaaS 的生存危机** — 当 AI 能直接按需求生成软件时，没有 headless 模式的产品将面临淘汰风险

### 🔧 Claude Code 新功能：/ultraplan

Anthropic 的 **Thariq** 宣布 Claude Code 推出 `/ultraplan` 模式——Claude 在云端为你构建完整的实现计划，你可以在线阅读、编辑，然后选择在云端或终端执行。与 plan 模式使用相近的 token 额度。

> "实现有时需要本地环境和交互，但规划可以在云端完成。"

🔗 [详情](https://x.com/trq212/status/2042671370186973589) ｜ [文档说明](https://x.com/trq212/status/2042671371684319723)

### 📝 Claude for Word 进入 Beta

Claude 正式登陆 Microsoft Word 侧边栏，支持直接在文档中起草、编辑、修改，保留原有格式并以修订模式呈现。更重要的是，**Claude for Word / Excel / PowerPoint 之间共享上下文**，实现跨文档协作。Team 和 Enterprise 计划可用。

🔗 [公告](https://x.com/claudeai/status/2042670341915295865)
