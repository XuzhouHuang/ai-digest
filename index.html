Marc Andreessen：浏览器之死、Pi + OpenClaw，以及「这次真的不一样」

节目： Latent Space

Andreessen 回顾了神经网络架构从争议到共识的七十年历程，将当下 AI 浪潮定义为「八十年一夜成名」——从 ChatGPT 到 o1 再到 OpenClaw，每一步都是数十年技术积累的爆发。他深入探讨了浏览器作为计算入口的衰落，以及 AI agent 如何重塑人机交互范式。

🐦 Builder 动态

🛠️ 开发工具与 Agent

  • @swyx — 「我们实现了 agentic 自我迭代」：把博文和推文直接丢给 Devin，它能一次性完成完整实现。Agent 写代码的时代真的来了。→

  • @ryolu_ — Cursor 发布全新界面和图标集，同时推出 Composer 2，周末双倍用量。设计感和实用性拉满。→

  • @steipete — GitHub API 频繁触发限流，「这根本没为 agent 场景设计」。基础设施跟不上 agent 时代的节奏。→

  • @garrytan — Garry’s List 页面性能跑到 95 分，发布 /plan-devex-review 工具帮助打造优秀的开发者体验。→

🤖 AI 产品与生态

  • @claudeai — Claude 全平台接入 Microsoft 365 连接器，Outlook、OneDrive、SharePoint 一键打通。企业级 AI 助手又进一步。→

  • @petergyang — 分享 OpenClaw soul.md 的个性化 prompt 技巧：「Warm, sharp, dry humor. Not a chatbot — a person.」让你的 AI 助手真正有人味。→

  • @zarazhangrui — 新趋势观察：人们开始把同事、KOL 甚至前任「蒸馏」成 agent skill。数字分身从概念走向现实。→

💡 观点与思考

  • @levie — 「人类认知有上限。即使你不审查 agent 的每个动作，你能同时 hold 住的信息量也是有限的。」AI 管理的本质是注意力分配。→

  • @steipete — Anthropic 的最新举动对生态不利，他和 @davemorin 尝试劝说延迟了一周。开放生态的博弈仍在继续。→

  • @nikunj — 继 token 焦虑之后，「Claude Code 眼」要成新病了 😅。Builder 们的职业病清单又多了一项。→
